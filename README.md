# Cocuk Oyun

Flutter scaffold for a preschool coloring app.

## Current Scope

- First SLC: ad-free coloring core for 4-5 year old children.
- Out of scope for now: mini game, ads, payments, photo-to-coloring AI, camera, gallery access.
- Product memory and active phase notes live in `brain/`.
